# Gather

Gather is a modern calendar and life organizer for families, work, and shared tables.

## Features
- Shared tables (family, work, personal)
- Recurring events
- Notes and memories
- Role-based sharing (viewer/editor)

## Tech Stack
- React + Vite
- Supabase Auth & Database
- Cloudflare Pages

## Development
```bash
npm install
npm run dev
